You already have an admin account for your jmatthew@clarkson.edu email, it will send a pin to your email. (No password)

## Website Guide

You should find yourself in the dashboard upon logging in. From here you can select a new page to go into. You can always return to the dashboard by clicking dashboard in the top left.

#### Sanctions & COIs

On this page you will see different rows of sanctions, clicking on one will dropdown more information, and show an edit and delete buttons. If the status is pending it will also show approve and deny buttons.

At the top right, there is a blue button to create a new sanction.

On the left bar there is a sanctions tab, which you can use to instantly get to this page, clicking request a sanction will immediately open the create a sanction page.

All these features should be working.

#### Clubs

This page will list rows of different clubs, hovering over a clubs name allows you to edit it.

In the top right you can create a new club.

All these features should be working.

#### Sites & Buildings

This page will list sites, which will then list all buildings belonging to them. Hovering over a site name allows you to edit the site. Hovering over a building name allows you to edit the building.

In the top right you can create a new site. In the top right of a site, you can create a new building.

There is a filter at the top, that will expand when clicked on. This will allow you to filter (sites) based on their state.

All these features should be working.

#### Users

Same as clubs, pretty much.

This page will list rows of different users, hovering over a user name allows you to edit it.

In the top right you can create a new user.

All these features should be working.

#### Incident Reports

Same as sanctions, pretty much.

On this page you will see different rows of Incident reports, clicking on one will dropdown more information, and show an edit and delete buttons. 

At the top right, there is a blue button to create a new incident.

Uploading attachments does not currently work, but everything else should work.
